\# 32-bit Pipelined CPU with Cache



\## Overview

This project implements a 32-bit pipelined CPU with cache support using Verilog HDL.



\## Features

\- 32-bit pipelined CPU datapath

\- ALU, register file, and control unit

\- Forwarding unit

\- Hazard detection unit

\- Instruction and data memory

\- Data cache module

\- FPGA-ready RTL structure



\## Tools Used

\- Verilog HDL

\- ModelSim

\- Vivado
